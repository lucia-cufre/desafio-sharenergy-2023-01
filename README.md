# Desafio para o processo seletivo SHARENERGY 2023/01

Como iniciar o projeto?

Back-end: 
1. cd backend
2. npm run dev

Front-end:
1.cd frontend
2.npm start

### Ferramentas e Stack utilizadas

- Frontend: ReactJS, MaterialUI, Typescript, HTML e CSS
- Backend: NodeJS, MongoDB, Mongoose, Typescript
